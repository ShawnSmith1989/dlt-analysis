# 大乐透数据更新指南

## 问题描述
当点击"更新数据"按钮时，如果显示"浏览器安全限制"错误，通常是因为访问方式不正确。

## 解决方案

### 1. 正确的访问方式

**不要直接双击打开HTML文件**，这样会因为浏览器安全策略导致功能无法使用。

**正确的访问方式是：**

1. 启动本地服务器：
   ```
   方法一：双击运行 start_server.bat
   方法二：在命令行中执行 python start_server.py
   ```

2. 在浏览器中访问：
   ```
   http://localhost:8888/chart.html
   ```

### 2. 更新数据功能

点击"更新数据"按钮后，会弹出模态框，包含三个标签页：

1. **更新最新数据**：获取最新的开奖数据
2. **补齐缺失数据**：检查并补齐历史缺失数据
3. **手动添加数据**：手动输入开奖数据

### 3. 如果自动更新失败

如果自动更新失败，可以：

1. 使用手动添加数据功能：
   - 切换到"手动添加数据"标签页
   - 填写期号、日期和号码
   - 点击"添加数据"按钮

2. 使用命令行手动更新：
   - 打开命令提示符
   - 执行以下命令：
     ```
     cd d:\daletgou\dlt4
     node update_data.js  # 更新最新数据
     或
     node update_missing_data.js  # 补齐缺失数据
     ```
   - 刷新浏览器页面

### 4. 常见问题

**Q: 为什么不能直接打开HTML文件？**
A: 浏览器的安全策略限制了网页直接访问本地文件系统，这会导致AJAX请求失败。

**Q: 服务器启动失败怎么办？**
A: 确保已安装Python，并且8888端口没有被其他程序占用。

**Q: 更新数据后页面没有变化？**
A: 尝试强制刷新页面（Ctrl+F5）或清除浏览器缓存。
