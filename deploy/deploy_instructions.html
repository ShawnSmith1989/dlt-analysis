EdgeOne Pages 部署指南

由于环境变量配置问题，我们提供以下两种部署方法：

方法一：手动上传到EdgeOne Pages

步骤：
  1. 登录 腾讯云EdgeOne控制台
  2. 创建新项目或选择现有项目
  3. 上传以下文件：
    • chart.html
    • index.html
    • chart.js
    • data.js
  4. 设置默认首页为 chart.html
  5. 发布并获取访问URL

方法二：使用其他静态托管服务

推荐平台：
  • GitHub Pages - 免费且易于使用
  • Vercel - 优秀的静态网站托管
  • Netlify - 功能强大的静态网站平台

上传deploy目录中的所有文件即可。

注意事项
  • 这是静态部署版本，仅支持手动添加数据功能
  • 手动添加的数据仅在当前会话有效，刷新页面后会丢失
  • 如需永久保存数据，请定期更新data.js文件并重新部署
