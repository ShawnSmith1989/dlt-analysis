# AI大乐透号码生成功能 - 需求分析文档

## 一、功能概述

### 1.1 功能名称

AI智能号码生成器（基于五行理论的号码推荐系统）

### 1.2 功能描述

用户输入时间、天气、地理位置、心情等自然地理数据，系统根据中国传统五行理论（金木水火土），将这些因素与彩票号码的五行属性进行匹配，生成一组推荐号码。

### 1.3 核心价值

- 融合传统五行智慧与现代数据分析
- 提供个性化的号码推荐体验
- 增加用户参与感和趣味性

## 四、算法设计

### 4.1 核心算法流程

```
┌─────────────────────────────────────────────────────────────┐
│                    AI号码生成算法                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 收集用户输入数据                                         │
│         ↓                                                   │
│  2. 计算各参数对应的五行权重                                  │
│         ↓                                                   │
│  3. 综合计算五行得分                                         │
│         ↓                                                   │
│  4. 根据五行得分选择号码区间                                  │
│         ↓                                                   │
│  5. 在各区间内随机选取号码                                    │
│         ↓                                                   │
│  6. 输出推荐号码组合                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 五行权重计算公式

```javascript
// 各参数权重配置
const WEIGHTS = {
    time: 0.20,      // 时间（时辰）
    weather: 0.25,   // 天气综合
    location: 0.15,  // 地理位置
    mood: 0.20,      // 心情状态
    lucky: 0.20      // 幸运数字
};

// 五行得分计算
function calculateElementScore(inputs) {
    let scores = { 金: 0, 木: 0, 水: 0, 火: 0, 土: 0 };
    
    // 时间贡献
    scores[getTimeElement(inputs.hour)] += WEIGHTS.time;
    
    // 天气贡献（温度、湿度、风向等加权平均）
    scores[getTempElement(inputs.temp)] += WEIGHTS.weather * 0.3;
    scores[getHumidityElement(inputs.humidity)] += WEIGHTS.weather * 0.2;
    scores[getWindElement(inputs.windDir, inputs.windLevel)] += WEIGHTS.weather * 0.3;
    scores[getMoonElement(inputs.moonPhase)] += WEIGHTS.weather * 0.2;
    
    // 地理贡献
    scores[getLocationElement(inputs.province)] += WEIGHTS.location;
    
    // 心情贡献
    scores[getMoodElement(inputs.mood)] += WEIGHTS.mood;
    
    // 幸运数字贡献
    scores[getLuckyElement(inputs.luckyNumber)] += WEIGHTS.lucky;
    
    return scores;
}
```

### 4.3 号码选择算法

```javascript
function generateNumbers(elementScores) {
    // 1. 按五行得分排序
    let sortedElements = Object.entries(elementScores)
        .sort((a, b) => b[1] - a[1]);
    
    // 2. 获取前3个五行属性作为主要选号依据
    let topElements = sortedElements.slice(0, 3);
    
    // 3. 根据五行属性获取号码池
    let frontPool = [];  // 前区号码池
    let backPool = [];   // 后区号码池
    
    topElements.forEach(([element, score]) => {
        frontPool.push(...getFrontNumbersByElement(element));
        backPool.push(...getBackNumbersByElement(element));
    });
    
    // 4. 随机选取前区5个号码
    let frontNumbers = shuffleAndPick(frontPool, 5);
    
    // 5. 随机选取后区2个号码
    let backNumbers = shuffleAndPick(backPool, 2);
    
    return { frontNumbers, backNumbers };
}
```

### 6.2 主要函数
